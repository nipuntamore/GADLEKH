# GADLEKH: Graphical Archive for Data, Legacy, Exploration, and Kingdom Heritage

**GADLEKH** is a Java-based desktop application designed to document and explore the strategic architecture and history of the forts built or fortified by **Chhatrapati Shivaji Maharaj**. The project serves as an interactive archive, providing users with detailed historical insights, geographical data, and a knowledge-testing quiz module.

## 🚀 Features

  * **Interactive Fort Explorer:** A sidebar navigation to browse through various forts like Raigad, Shivneri, and Gingee.
  * **Dynamic Information Display:** Real-time updating of descriptions and high-quality images based on user selection.
  * **Historical Archive:** Detailed data on elevation, key landmarks, strategic significance, and architectural founders.
  * **Knowledge Quiz:** An integrated MCQ module to test the user's understanding of Maratha history.
  * **Modern UI:** Utilizes the `Nimbus` Look and Feel for a sleek, dark-themed aesthetic with Maratha-inspired orange accents.

## 🛠️ Technology Stack

  * **Language:** Java
  * **Framework:** Java Swing (GUI)
  * **Layout Management:** `BorderLayout` and `FlowLayout`
  * **Data Structures:** `Vector` and `DefaultListModel`
  * **Styling:** Custom RGB color schemes and `Nimbus` UI Manager

## ⚙️ How to Run

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/nipuntamore/GADLEKH.git
    ```
2.  **Navigate to the Directory:**
    ```bash
    cd GADLEKH/src
    ```
3.  **Compile the Program:**
    ```bash
    javac GadlekhApp.java
    ```
4.  **Run the Application:**
    ```bash
    java GadlekhApp
    ```

## 📜 Historical Significance

The forts of the Maratha Empire are a testament to military engineering and strategic foresight. From the "Gibraltar of the East" (**Raigad**) to the "Troy of the East" (**Gingee**), this application aims to preserve the legacy of *Swarajya* through a digital medium.
